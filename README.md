# Saurav Upreti

I'm a software engineer based in Kathmandu, Nepal, working mostly on the backend and increasingly on applied AI. By day I build backend systems at Renegade Insurance. On the side, I run [finduni.ai](https://finduni.ai) — an AI-powered study-abroad platform for Nepali students — where I work across the whole stack (React, Supabase, Vercel) and the AI features layered on top of it.

I started out in mobile development with Flutter, moved into backend engineering, and these days most of my curiosity goes into building reliable systems around LLMs — agents, retrieval, evaluation, and the unglamorous production plumbing that makes them actually work. A lot of what's on this profile is me learning that in the open.

A few things I've built recently:

- **[careerlens-mcp](https://github.com/upretisaurav/careerlens-mcp)** — an MCP server that gives Claude real-time career intelligence: salary benchmarks, live job listings, skill-demand analysis, and resume ATS scoring. Built with FastMCP + FastAPI + React.
- **[gemini-agent](https://github.com/upretisaurav/gemini-agent)** — a small autonomous agent on Google's Gemini with function calling that can read, write, and run files inside a sandboxed directory.
- **[systemroast](https://github.com/upretisaurav/systemroast)** — point it at your system architecture and it roasts it back to you in Gen Z slang. No login, just pain.
